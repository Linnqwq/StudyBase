# MVP 文档

> 开发内部文档 · v0.1

---

## 我们在做什么

一个面向大学生的学习资料管理工具。学生上传课程文件，系统自动整理、提取关键信息、生成摘要，并在课程空间里统一展示。

---

## 功能范围

### 做

| 功能 | 说明 |
|---|---|
| 课程空间创建 | 用户为每门课创建独立空间，作为资料统一入口 |
| 文件上传 | 支持课程大纲、课件、论文、笔记、作业（PDF、PPT、Word） |
| 自动分类 | 根据文件内容（非文件名）判断类型，自动归类 |
| 关键信息提取 | 从课程大纲提取成绩比例、截止日期、考试信息、课程政策 |
| 摘要生成 | 对课件、论文生成结构化摘要 |
| 课程目标总览板 | 人工智能根据上传文件自动生成主目标面板，展示课程核心目标、近期截止日期、成绩构成 |
| 课程页面展示 | 将以上所有信息整合成结构清晰的课程页面 |

### 不做（v1）

| 功能 | 原因 |
|---|---|
| 移动端 | 先验证 Web 端 |
| 多人协作 | 不在核心场景内 |
| 日历集成 | 截止日期列表已够用 |
| 对话式辅导 | 不同产品方向 |
| 全格式 OCR | 仅支持可提取文字的文件 |
| 学习方法推荐 | 后续版本功能 |

---

## 用户流程

```
创建课程空间
    ↓
上传课程文件
    ↓
系统自动分类 + 解析内容
    ↓
人工智能提取关键信息 + 生成摘要 + 生成目标总览板
    ↓
课程页面展示所有结果
```

## 数据模型（初步）

```
用户（User）
└── 课程（Course）
    └── 文件（Document）
        ├── 提取结果（ExtractedInfo）
        │   ├── 成绩比例
        │   ├── 截止日期
        │   └── 考试信息
        ├── 摘要（Summary）
        └── 分块记录（Chunk）— 后续语义检索用
```
