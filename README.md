# course-organizer-ai

An AI-powered study document organizer for students.

## Goal
Help students upload and organize course materials such as syllabi, slides, and readings.

## Core features
- Upload course documents
- Extract key information such as grading, deadlines, and course policies
- Organize documents by course
- Generate summaries and study structure

## Project structure
- frontend: web interface
- backend: API, document processing, AI pipeline
- docs: project planning and technical notes
